# Live Photos Directory

## Purpose
This directory stores iPhone Live Photos uploaded by clients for video production services.

## File Types Supported
- **HEIC/HEIF** - iPhone Live Photo still images
- **MOV** - iPhone Live Photo motion videos
- **JPG/JPEG** - Exported still frames from Live Photos

## Business Integration
Live Photos are perfect source material for:
- VideoGrams (3-second enhanced clips)
- Social media content creation
- Memory preservation videos
- Family milestone documentation

## Processing Workflow
1. Client uploads Live Photos via web interface
2. Files are timestamped and stored securely
3. Production team can access both still and motion components
4. Convert to professional video content using DreamFrame services

## Technical Notes
- Files are automatically renamed with timestamps for organization
- Supports batch uploads for multiple Live Photos
- Compatible with iPhone-generated HEIC and MOV formats
- Secure filename handling prevents conflicts
