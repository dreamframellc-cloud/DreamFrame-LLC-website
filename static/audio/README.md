# Audio Files for DreamFrame LLC

## Background Music
- **futuristic-ambient.mp3** - Contemporary futuristic background music
- **futuristic-ambient.ogg** - OGG format for broader browser compatibility

## Usage
The background music system includes:
- Floating music toggle button (bottom-right corner)
- Auto-start after first user interaction (browser requirement)
- Volume set to 30% for non-intrusive experience
- Responsive design for mobile devices
- Smooth fade animations and visual feedback

## Audio Requirements
- Format: MP3 and OGG for cross-browser compatibility
- Volume: Optimized for background ambiance
- Loop: Seamless looping for continuous experience
- Size: Compressed for fast loading

## Implementation
The music player is integrated into the homepage with:
- CSS animations and hover effects
- JavaScript controls for play/pause
- Session storage to prevent auto-restart
- Accessible controls with clear visual states
